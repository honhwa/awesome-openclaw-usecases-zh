# 全网比价助手

在淘宝、京东、拼多多、抖音、快手之间来回切换比价，手动找优惠券，盯限时折扣——这些重复操作每次购物都要花半小时以上。

这个用例让 OpenClaw 帮你跨平台搜索商品、对比价格、查找优惠券，一句话搞定全网比价。

## 它能做什么

- **全网商品搜索**：一句话搜索淘宝/天猫、京东、拼多多、抖音、快手的商品
- **价格对比**：同一商品在不同平台的价格一目了然
- **优惠券查找**：自动查找可用的优惠券和折扣
- **商品详情**：获取商品详情和直达购买链接

支持平台：

| 平台代码 | 平台 |
|:--------:|------|
| 1 | 淘宝 / 天猫 |
| 2 | 京东 |
| 3 | 拼多多 |
| 7 | 抖音 |
| 8 | 快手 |

## 所需技能

[maishou（买手）](https://playbooks.com/skills/openclaw/skills/maishou) —— OpenClaw 官方技能库中的购物助手技能。

## 如何设置

1. 安装买手技能：
```bash
npx playbooks add skill openclaw/skills --skill maishou
```

2. 直接用自然语言提问：
```text
帮我搜一下 AirPods Pro 2，对比淘宝和京东的价格，找最便宜的那个。
```

3. 更多用法：
```text
帮我搜一下戴森吹风机在拼多多上有没有优惠券。

对比一下这款键盘在五个平台的价格，列个表格出来。

帮我看看这个商品链接在其他平台有没有更便宜的。
```

## 实用建议

- **大促前用**：双十一、618 前用这个对比各平台真实底价，避免"先涨后降"
- **凑单场景**：让 OpenClaw 帮你在同一平台内找凑单商品达到满减门槛
- **日常快速决策**：不确定在哪买的时候，一句话出结果
